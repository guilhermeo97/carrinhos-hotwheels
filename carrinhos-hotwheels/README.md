# CRUD Carrinhos HotWheels Next.js

Este é um projeto front-end criado com React, Next.js, TypeScript e Tailwind CSS.

## Pré-requisitos

- Node.js (versão 14.x ou superior)
- npm ou Yarn

## Instalação

```bash
git clone https://github.com/usuario/meu-projeto-next.git
cd meu-projeto-next
npm install
# ou
yarn install
```

## Execução

```bash
npm run dev
# ou
yarn dev
```
